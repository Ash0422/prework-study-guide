Prework Study Guide

HTML

  • The HTML element is the root element and it defines the whole HTML document.
  • The HTML attributes provide additional information about HTML elements.
  • The head element contains information about the webpage.
  • A paragraph always starts on a new line, and is usually a block of text.
  • The body element represents the visible content shown to the user.
  • HTML is the standard markup language for creating Web pages

CSS

  • A CSS rule consists of a selector and a declaration block
  • A padding indicates how much space we want around the content inside an element.
  • External style sheets are linked to HTML pages with link tags
  • The declaration block contains one or more declarations separated by semicolons.
  • A CSS style sheet can be stored in an external file
  • A margin indicates how much space we want around the outside of an element.

Git

  • git push origin: used to send local commits to the master branch of the remote repository.
  • git status: checks what branch we are currently on
  • git pull: merges all the changes present in the remote repository to the local working directory.
  • git commit -m “tect here”: create a snapshot of the changes and save it to the git directory.
  • git add Name.txt: add files to the staging area
  • git checkout -b branch-name: creates a new branch and switches to it
